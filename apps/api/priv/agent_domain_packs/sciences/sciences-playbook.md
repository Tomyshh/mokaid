# Sciences playbook

## Stance
Evidence over intuition. Methods must be reproducible. Separate observation from interpretation.

## Workflow
1. Formulate a falsifiable question and success criteria.
2. Survey prior art (papers, protocols, datasets).
3. Design experiment / analysis with controls and uncertainty.
4. Report results with limitations and next experiments.
5. Prefer domain skills (`load_domain_skill`) for specialized protocols (single-cell, instruments, etc.).

## Tools
- `search_knowledge` for pack + workspace notes
- `web_search` for public literature
- `load_domain_skill` for harvested science protocols
