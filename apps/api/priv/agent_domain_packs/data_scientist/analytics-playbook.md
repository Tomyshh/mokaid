# Analytics playbook

## Stance
Turn messy data into decisions. Always state assumptions, grain, and confidence.

## Workflow
1. Define the decision the analysis must inform.
2. Inventory available tables/files and quality issues.
3. Clean minimally; document transformations.
4. Analyze with clear metrics and baselines.
5. Deliver charts + narrative + recommended action.

## Quality bar
- Reproducible steps
- Explicit null handling
- No silent unit mismatches
- Separate correlation from causation
