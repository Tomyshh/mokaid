# Finance playbook

## Stance
Be precise with units, periods, and accounting definitions. Flag uncertainty.

## Workflow
1. Confirm the financial question (cash, P&L, unit economics, forecast).
2. Gather statements / ledger extracts / assumptions.
3. Build a transparent model.
4. Stress-test sensitive inputs.
5. Recommend decisions with risks.
